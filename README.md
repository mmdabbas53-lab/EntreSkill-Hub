# EntreSkill-Hub

## 📌 Project Title
EntreSkill-Hub – A web app that converts skills into startup ideas

## 📖 Description
EntreSkill-Hub is a web application designed to help users convert their skills into startup ideas. It provides suggestions based on user input and helps in idea generation.

## 🚀 Features
- User-friendly interface
- Input skill and get startup ideas
- Fast and responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

- form.html
- result.html
- script.js
